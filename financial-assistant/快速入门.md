## 项目简介

本系统是一个基于LLM的**智能财报问答平台**，能够：

- **Text2SQL**：将自然语言问题自动转换为SQL查询
- **智能对话**：多轮对话理解财务问题并给出答案
- **推理调试**：可视化AI的思考过程
- **图表生成**：自动生成数据可视化图表

### 核心技术栈

| 组件 | 技术 |
|------|------|
| LLM | OpenAI GPT、Gemini、DeepSeek、本地vLLM |
| 数据库 | PostgreSQL（结构化数据）+ ChromaDB（向量检索） |
| 前端 | Streamlit Web界面 |
| Embedding | OpenAI text-embedding-3-small 或本地BGE模型 |

## 3分钟快速启动

### 前提条件

- [ ] **Python 3.10+** 已安装
- [ ] **Docker Desktop** 已安装并运行
- [ ] **API密钥**（如果队友已配置好 `.env` 文件则跳过）
  - 推荐 DeepSeek：https://platform.deepseek.com/
  - 多个队友可以共用同一个密钥

### 步骤1：安装依赖

```bash
cd c:\cursor_Xiangmu\taidi_cup_B\financial-assistant\chatbot_financial_statement

# 创建虚拟环境
conda create -y -n text2sql python=3.10
conda activate text2sql
# 或使用 venv
python -m venv venv
venv\Scripts\activate  # Windows

# 安装依赖（使用国内镜像）
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 步骤2：检查.env文件（已有配置则跳过）

> ⚠️ **如果你拉取的是队友配置好的项目，这步可能已经完成！**

1. **检查是否已有 `.env` 文件**：
   ```bash
   # 查看是否有.env文件
   dir .env  # Windows
   ls -la .env  # Mac/Linux
   ```

2. **如果有 `.env` 且内容正确** → 直接跳过步骤2和3，进入步骤4

3. **如果没有或需要修改**，在 `chatbot_financial_statement/` 目录创建 `.env` 文件：

```ini
# ========== LLM配置（必须填写） ==========
OPENAI_API_KEY=sk-your-api-key-here
LLM_HOST=https://api.deepseek.com

# ========== 数据库配置（保持默认） ==========
DB_NAME=financial_db
DB_USER=postgres
DB_PASSWORD=postgres
DB_HOST=localhost
DB_PORT=5432

MONGO_DB_HOST=localhost
MONGO_DB_USER=admin
MONGO_DB_PASSWORD=admin
MONGO_DB_PORT=27017

# ========== Embedding配置 ==========
LOCAL_EMBEDDING=False
EMBEDDING_SERVER_URL=https://api.openai.com/v1
EMBEDDING_MODEL=text-embedding-3-small

# ========== Reranker配置（可选） ==========
RERANKER_SERVER_URL=http://localhost:8081
RERANKER_MODEL=BAAI/bge-reranker-v2-m3
```

**获取API密钥**：
- **DeepSeek**（推荐）：https://platform.deepseek.com/ → API Keys → 创建密钥
- **OpenAI**：https://platform.openai.com/ → API Keys

> ⚠️ 注意：
> 1. **如果项目已有 `.env` 文件且配置正确，跳过此步骤**
> 2. `.env` 文件必须在 `chatbot_financial_statement/` 目录下（与 `home.py` 同级）
> 3. 文件名是 `.env` 不是 `.env.txt`（Windows需显示文件扩展名）
> 4. 多人可以共用同一个API密钥

### 步骤3：启动系统

```bash
# 1. 启动数据库
docker-compose up -d

# 2. 等待5秒
timeout /t 5  # Windows
sleep 5       # Mac/Linux

# 3. 验证容器状态
docker ps
# 应看到 postgres2 和 mongodb2

# 4. 初始化数据（首次运行，5-10分钟）
python setup.py --preprocess v3.2 --force True --openai True --vectordb chromadb

# 5. 测试环境
python test.py

# 6. 启动Web界面
streamlit run home.py
```

✅ 浏览器自动打开 `http://localhost:8501`

## 常见问题与解决

### Q1：Docker容器启动失败

**现象**：`docker ps` 看不到容器

```bash
# 查看错误日志
docker-compose logs

# 重启容器
docker-compose down
docker-compose up -d

# 检查端口占用
netstat -an | findstr "5432"  # Windows
```

### Q2：PostgreSQL连接报错（编码错误）

**现象**：`'utf-8' codec can't decode byte 0xd6`

**解决**：重建数据库为GBK编码

```bash
# 1. 停止容器
docker-compose down

# 2. 删除数据卷
docker volume rm financial-assistant_postgres_data

# 3. 修改 docker-compose.yml 中的环境变量
POSTGRES_INITDB_ARGS: "--encoding=GBK"

# 4. 重启
docker-compose up -d
```

### Q3：MongoDB认证失败

**现象**：`Authentication failed`

**解决**：彻底重建

```bash
docker-compose down
docker volume rm financial-assistant_mongo_data
docker-compose up -d
```

### Q4：Embedding API调用失败

**解决**：改用OpenAI Embedding

```ini
LOCAL_EMBEDDING=False
EMBEDDING_SERVER_URL=https://api.openai.com/v1
EMBEDDING_MODEL=text-embedding-3-small
```

### Q5：.env文件找不到

**原因**：
1. 文件位置不对（必须在 `chatbot_financial_statement/` 目录）
2. 文件命名为 `.env.txt`（Windows隐藏了扩展名）

**解决**：
1. 确认文件位置：`c:\cursor_Xiangmu\taidi_cup_B\financial-assistant\chatbot_financial_statement\.env`
2. Windows显示文件扩展名：资源管理器 → 查看 → 勾选"文件扩展名"

### Q6：streamlit不是内部命令

**解决**：

```bash
# 激活虚拟环境
venv\Scripts\activate

# 验证安装
pip show streamlit

# 如未安装
pip install streamlit
```

### Q7：端口8501被占用

```bash
# 查看占用进程
netstat -ano | findstr :8501

# 杀死进程
taskkill /PID <PID> /F

# 或更换端口
streamlit run home.py --server.port 8502
```

### Q8：setup.py运行失败

**可能原因**：
1. API密钥无效 → 检查 `.env`
2. 网络问题 → 测试：`curl https://api.deepseek.com/v1/chat/completions`
3. 内存不足 → 使用小数据版本 `--preprocess v3`
4. 磁盘空间不足 → 需要至少2GB

## 附件4任务执行

附件4问题是：**"茅台2022年的营业收入是多少？"**

### 方式A：Web界面（推荐）

1. 打开 `http://localhost:8501`
2. 点击 **Text2SQL** 页面
3. 输入：`茅台2022年的营业收入`
4. 查看SQL和结果

### 方式B：批量脚本

```bash
python batch_runner.py \
  --input_file 附件4.xlsx \
  --output result_2.xlsx \
  --generate_chart
```

输出：
- `result_2.xlsx` - 答案文件
- `charts/` - 图表目录

### 方式C：直接调用API

```python
from chatbot_financial_statement.agent.text2sql import Text2SQL
from chatbot_financial_statement.agent.const import Text2SQLConfig

config = Text2SQLConfig(
    llm_api_key="your-api-key",
    llm_base_url="https://api.deepseek.com"
)
t2s = Text2SQL(config)
result = t2s.query("茅台2022年的营业收入是多少？")
print(result)
```
