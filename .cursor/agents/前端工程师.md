## Streamlit常用模式

```python
import streamlit as st

st.set_page_config(page_title="智能财报问答", page_icon="🏦")
st.title("💬 财报问答助手")

# 侧边栏
with st.sidebar:
    st.header("设置")
    api_key = st.text_input("API Key", type="password")

# 主内容区
col1, col2 = st.columns(2)
with col1:
    st.metric("查询次数", "1,234")
```
