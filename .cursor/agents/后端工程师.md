## LLM调用示例

```python
from llm.llm.openai_compatible import LLMClient

client = LLMClient(
    api_key="your-api-key",
    base_url="https://api.deepseek.com"
)
response = client.generate("你好，请介绍自己")
```

## Text2SQL使用示例

```python
from chatbot_financial_statement.agent.text2sql import Text2SQL
from chatbot_financial_statement.agent.const import Text2SQLConfig

config = Text2SQLConfig(
    llm_api_key="your-api-key",
    llm_base_url="https://api.deepseek.com"
)
t2s = Text2SQL(config)
result = t2s.query("茅台2022年的营业收入是多少？")
```
